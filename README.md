# Data Governance

Полный курс по управлению данными — от метаданных и каталогов до приватности, качества и контроля доступа.

## О курсе

- **Автор:** Lev Neganov
- **Уровень:** Intermediate
- **Длительность:** ~35 часов
- **Язык:** Русский

## Модули

0. **Введение в курс** — обзор курса, целевая аудитория и структура
1. **Основы Data Governance** — определения, DAMA-DMBOK, бизнес-ценность
2. **Архитектура и Моделирование Данных** — модели данных, архитектурные паттерны
3. **Метаданные и Каталоги Данных** — метаданные, OpenMetadata, data lineage
4. **Качество Данных и Observability** — Great Expectations, метрики качества
5. **Приватность и Compliance** — GDPR, 152-ФЗ, PII, анонимизация
6. **Безопасность и Контроль Доступа** — RBAC, ABAC, шифрование
7. **Внедрение Программы Governance** — организационная структура, KPI
8. **AI Governance** — управление ML-моделями, fairness, explainability
9. **Экосистема Инструментов** — обзор инструментов DG
10. **Capstone Project** — финальный проект
