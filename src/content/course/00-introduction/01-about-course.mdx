# О курсе Data Governance

Добро пожаловать в курс **Data Governance** — комплексное руководство по управлению данными как стратегическим активом организации.

## Для кого этот курс

Курс предназначен для дата-инженеров, аналитиков данных, руководителей данных (CDO), специалистов по compliance и всех, кто участвует в построении культуры управления данными.

## Структура курса

Курс состоит из 11 модулей, охватывающих все ключевые области Data Governance согласно методологии DAMA-DMBOK:

- **M00** — Введение в курс
- **M01** — Основы Data Governance
- **M02** — Архитектура и Моделирование Данных
- **M03** — Метаданные и Каталоги Данных
- **M04** — Качество Данных и Observability
- **M05** — Приватность и Compliance
- **M06** — Безопасность и Контроль Доступа
- **M07** — Внедрение Программы Governance
- **M08** — AI Governance
- **M09** — Экосистема Инструментов
- **M10** — Capstone Project
